安装git

git是版本控制工具

人话翻译:协同开发的时候，存代码的

可以存到github，gitee（码云）

git（/ɡɪt/[3]]， 音频 帮助·信息，吉特）是一个分布式版本控制软件，最初由林纳斯·托瓦兹创作，于2005年以GPL许可协议发布。最初目的是为了更好地管理Linux内核开发而设计。应注意的是，这与GNU Interactive Tools[4]（一个类似Norton Commander界面的文件管理器）不同。

2005年，安德鲁·垂鸠写了一个简单程序，可以连接BitKeeper的仓库，BitKeeper著作权拥有者拉里·麦沃伊认为安德鲁·垂鸠对BitKeeper内部使用的协议进行逆向工程，决定收回无偿使用BitKeeper的许可。Linux内核开发团队与BitMover公司进行磋商，但无法解决他们之间的歧见。林纳斯·托瓦兹决定自行开发版本控制系统替代BitKeeper，以十天的时间编写出git第一个版本[12][13]。

https://zh.wikipedia.org/wiki/Git

git的出生于Linux之父之手，最开始，因为linux是开源的，所以全世界的人都能一起来维护这个项目把蛋糕做大，但是BitKeeper这个也是协同开发的工具，不让Linux之父用了，大佬一气之下，十天写出了git的第一版。别人十天写个项目，我十天只能摸十天鱼

第一步

安装好右键应该会有git bash here

输入ssh即可查看是否安装SSH

如上图所示，即已安装 SSH. 然后，输入 ssh-keygen -t rsa 命令（注意空格），表示我们指定 RSA 算法生成密钥，然后敲四次回车键，之后就就会生成两个文件，分别为秘钥 id_rsa 和公钥 id_rsa.pub. ***（注意：git中的复制粘贴不是 Ctrl+C 和 Ctrl+V，而是 Ctrl+insert 和 Shift+insert.）***文件的位置在 Git Bash 上面都有显示，默认生成在以下目录：

  • Linux 系统：~/.ssh
  • Mac 系统：~/.ssh
  • Windows 10 ：C:/Users/ASUS/.ssh

移动到对应的目录下，cat查看秘钥ssh-rsa，复制这段代码，扔到github里的ssh里即可

冷知识：

rsa是非对称加密的，一个公钥，一个秘钥，公钥直接公开到网络上，秘钥留给自己，他俩本质是俩比较大的质数

对面的人用公开的公钥加密，然后只有用秘钥才能解开，想暴力破解，必须得用分解大质数的方法试出来。所以这个方法被用的最多，比特币挖矿也是这个原理，cpu喜欢复杂的逻辑计算，显卡喜欢愣头青的莽，所以矿老板买一堆显卡，一个一个数字的试

所以黎曼猜想了解一下，分解大质数问题一旦被破解出来，世界上很多银行，国防，需要加密的地方都会直接裂开、

（量子计算机等价于超高配版cpu+显卡）
