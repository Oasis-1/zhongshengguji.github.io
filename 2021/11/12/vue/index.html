Vue-iec

html 略

css

  • SASS:基于Ruby，通过服务端处理，功能强大。解析效率高。需要学习Ruby语言，上手难度高于LESS。
  • LESS:基于NodeJS，通过客户端处理，使用简单。功能比 SASS简单，解析效率也低于SASS，但在实际开发中足够了，所以我们后台人员如果需要的话，建议使用LESS。

js

  • jq
  • Angular：谷歌的，java程序员开发的，特点是将后端的MVC模式搬到前端了
  • react：facebook的，提出了虚拟dom的概念，在内存中模拟dom的操作
  • vue：集合他俩的优点
  • axios：前端通信框架，因为vue的边界很明确

ui框架

  • Ant-Design:阿里巴巴出品，基于React的UI框架
  • ElementUl、iview、ice:饿了么出品，基于Vue 的UI框
  • Bootstrap: Twitter推出的一个用于前端开发的开源工具
  • AmazeUl:又叫“妹子UI”，一款HTML5跨屏前端框架

js构建工具

  • Babel:JS 编译工具，主要用于浏览器不支持的ES新特性，比如用于编译TypeScript
  • WebPack:模块打包器，主要作用是打包、压缩、合并及按序加载

nodejs的作者已经准备放弃了，正在开发一个deno，可能是太笨重了
