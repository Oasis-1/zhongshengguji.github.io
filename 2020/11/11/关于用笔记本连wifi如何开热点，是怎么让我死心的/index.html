关于用笔记本连wifi如何开热点，是怎么让我死心的

封装1809W10系统心得

https://user.qzone.qq.com/1939212190/blog/1605080953

详细教程

第一个问题是因为自己笔记本的ip地址无法更改 如果更改就弹出东西无法保存
直接把弹出来的内容给扔到百度里
最后使用ccleaner清除注册表后成功解决
本来以为是这个问题没法开热点

事实还是我太年轻了
netsh wlan set hostednetwork mode=allow ssid= key=用户密码都设置好了
netsh wlan start hostednetwork 这行的时候提示“无法启动承载网络。组或资源的状态不是执行请求操作的正确状态。”

然后我又顺藤摸瓜，把那段扔百度里，扔知乎里
有说网卡驱动的问题的，有说设备管理器里需要卸载的，各种各样
最后在一个犄角旮旯里找到最上面那个比较详细的
先用Type – NETSH WLAN show drivers
进行测试，看看网卡支不支持
然后再敲开关wifi的cmd

心得（时间成本14点~16点）
1.以后笔记本可以开热点不知道先来这里找下代码试一下了
2.找到一个批处理的开wifi的脚本

3.找到一个CCleaner激活码，直接扔百度
4.托不能改ip的福多学了一个
https://blog.csdn.net/zhaoyishi/article/details/85795884

netsh interface ip set dns name="以太网 3" static DNS地址

address

netsh interface ip set dns name="连接名称" source=dhcp
